## __13° Passo: Definindo a senha do seu SUPER USÚARIO__
> __Super Usúario (root) é o administrador maior em um sistema, possui permissão de fazer qualquer alteração no computador, por isso crie uma senha dificil, porém memorável!__
>
> Defina uma senha que você não esqueça pois muda-la é quase impossível, e ficar sem ela é se limitar a funções já disponiveis no windowns, um adendo importante __NÃO DEIXE A SENHA EM BRANCO__ isso significa para o sistema que você não precisa do ROOT, impossibilitando você (usúario) de acessar muitas ferramentas.
>
> ![alt text](image-13.png)
